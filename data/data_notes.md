# Data Notes

This repository shares aggregate data only.

## Analysis Basis

- Raw Google Forms submissions: 204.
- Retained exact response-content patterns: 181.
- Later exact-content duplicates removed: 23 submissions in five duplicate-content groups.
- Missing cells across the core model variables: 0.
- Invalid 1-7 values across the core model variables: 0.
- Scenario straightline patterns flagged for sensitivity review: 17; flagged, not excluded.

The exact-content rule is reproducible, but it was documented after data collection. The supplied materials did not contain a preregistered exclusion plan, attention checks, or a frozen exclusion log created before analysis.

## Interpretation

The sample was recruited through accessible online and school/social-network channels. It is a nonprobability convenience sample, so the results should not be treated as population estimates for all AI chatbot users.

The scenarios and checkpoints appeared in a fixed order. Scenario A and Scenario B also differed in total errors, error rate, timing, consistency, and wording. H2 therefore cannot isolate inconsistency as a causal factor.

Attention and intended actions were self-reported after hypothetical scenarios. The survey did not observe reading time, source checks, reliance, switching, or abandonment in a live chatbot task.
