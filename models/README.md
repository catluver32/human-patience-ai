# Rebuilt Models and Figures

This folder contains the rebuilt statistical audit workbook, public model-result tables, and descriptive figures used in the project.

## Primary Model Artifact

- [`AI_Frustration_Rebuilt_Models.xlsx`](AI_Frustration_Rebuilt_Models.xlsx) — complete audited workbook with data-quality checks, descriptives, H1-H4 models, scale diagnostics, counts, and source links.

## Public Model Tables

- [`hypothesis_summary.csv`](hypothesis_summary.csv) — evidence-aligned H1-H4 conclusions.
- [`h1_omnibus.csv`](h1_omnibus.csv) — Friedman tests and Kendall's W.
- [`h1_pairwise_wilcoxon.csv`](h1_pairwise_wilcoxon.csv) — Holm-adjusted paired contrasts.
- [`h2_results.csv`](h2_results.csv) — forced-choice and paired final-checkpoint results.
- [`h3_logistic_sensitivity.csv`](h3_logistic_sensitivity.csv) — attention-change logistic models.
- [`h4_logistic_sensitivity.csv`](h4_logistic_sensitivity.csv) — grouped switch-away logistic models.
- [`scale_reliability.csv`](scale_reliability.csv) — alpha and interpretation of the exploratory composite.

All models use the 181 retained response-content patterns created by the documented exact-content duplicate rule. The models describe associations within a nonprobability scenario survey; they do not establish causal effects or population prevalence.
